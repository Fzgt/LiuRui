## 17 Apr 两数之和
```python
class Solution(object):
    def twoSum(self, nums, target):
        length = len(nums)
        i = 0
        while i < length:
            j = i + 1
            while j < length:
                if nums[i] + nums[j] == target:
                    return [i, j]
                j += 1
            i += 1
```
